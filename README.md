# Dashboard-UI

A simple admin dashboard using the "PalaceSugar" e-commerce as an example. Features authentication, light/dark theme, and a system for creating and editing posts.

# Tech used

- **React/Next.js**
- **Shadcn/ui**
- **TypeScript**
- **TailwindCSS**
- **Recharts**
- **Lucide React**
- **React Hook Form & Zod**
